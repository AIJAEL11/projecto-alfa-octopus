## 🎯 What is Octopus AI?

Octopus AI is a **multi-module autonomous platform** that unifies 15+ AI-powered tools into a single cockpit. Designed for solopreneurs, small businesses, and marketing teams, it automates the entire pipeline from **lead capture → content creation → multi-channel distribution**.

Unlike point solutions that handle one task, Octopus orchestrates the full sales & marketing lifecycle through an intelligent agent system called **OCTOPUS** — a RAG 2.0+ conversational engine that understands context across all modules.

## ✨ Core Modules

| Module | Description |
|--------|-------------|
| **🐙 OCTOPUS (RAG 2.0+)** | Conversational AI agent with 5-layer retrieval (semantic memory, knowledge base, vector search, graph context, arms/integrations). Orchestrates all modules via natural language. |
| **🎨 Creative Studio** | AI-powered asset generation — images, videos, audio, and documents. Supports personalized lead-to-asset pipelines. |
| **📈 Growth Engine** | Smart prospecting with automated lead scoring, campaign creation, and multi-channel outreach. |
| **📺 Ad Factory** | Generate ready-to-publish ads for Facebook, Instagram, Google, and TikTok with AI-optimized copy and creatives. |
| **🎬 Motion Graphics** | AI video generation using Veo 3.1, Kling, and Seedance models. Text-to-video and image-to-video workflows. |
| **🤖 Sales Agent** | Embeddable AI sales agents with real-time lead capture, customizable personalities, and CRM integration. |
| **🗣️ Voice Agent** | Premium text-to-speech with ElevenLabs and real-time speech-to-text for voice-first workflows. |
| **🌉 Social Bridge** | Auto-publish content to 8+ social networks from a single interface. |
| **💻 Code Engine** | AI website generator with real-time preview, GitHub integration, and one-click deploy to Vercel/Netlify/Railway. |
| **🏭 Agent Factory** | Create, customize, and deploy autonomous AI agents with custom personalities and capabilities. |
| **🔧 Skill Factory** | Build reusable AI skills — modular capabilities that agents can execute autonomously. |
| **🧠 Ollama Chat** | Private local AI using Llama, Mistral, and DeepSeek models via Ollama integration. |
| **📋 Task Manager** | AI-enhanced task management with priority scoring, due dates, and intelligent categorization. |
| **📅 Smart Calendar** | AI-powered scheduling with natural language event creation. |
| **🧾 Quick Invoice** | Generate professional invoices with AI-assisted line items and client management. |
| **🏠 Smart Home (IoT)** | Voice and web control of Shelly IoT devices — smart home automation from the cockpit. |
| **🔌 API Hub** | Connect external APIs with visual configuration — no code required. |

## 🦾 Integrations (Brazos/Arms)

Octopus connects to external services through **Brazos** (Arms):

- **GitHub** — Push/pull repos, branches, commits, AI code review
- **Gmail** — Full read/write (list, search, send, draft, reply)
- **Google Workspace** — Calendar and Drive integration
- **Telegram** — Bot messaging and notifications
- **WhatsApp** — Business messaging via API
- **SMTP** — Custom email server integration
- **Ollama** — Local LLM models
- **Web3Forms** — Form submission handling
